Урок №1

1. Загрузил и установил VirtualBox.
2. Создал локальную машину, установил Ubuntu.
3. Установил VBox. Потребовалось дополнительно установить пакеты gcc make perl командой в терминале apt-dev install gcc make perl
4. Установил пакет openssh-server командой sudo apt-dev install openssh-server.
5. Установил Midnight Commander командой sudo apt-get install mc.
6. Установил подключение к серверу ubuntu через ssh - user@192.168.43.32 из терминала Windows и из Far.
7. Сгенерировал SSH ключ - ssh-keygen. Скопировал публичный ключ на сервере в файл .ssh/authorized_key.
8. Скопировал приватный ключ на Windows. Установил соединение по SSH ключу - ssh user@192.168.43.32 -i id_rsa.
9. Установил пакет ufw.
10. Настроил межсетевой экран на запрет всех входящих - sudo ufw default deny incoming и разрешение всех исходящих - sudo ufw default allow outgoing
11. Разрешил соединение SSH только по порту 22  - sudo ufw allow 22.
12. Запустил межсетевой экран - sudo ufw enable
13. Менял номер порта соединения SSH в файле sshd_config свойство Port. Соответственно соединялся - ssh user@192.168.43.32 -p XX
